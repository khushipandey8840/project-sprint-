# Cover Letter Generator

Ek simple aur user-friendly web application jo aapke liye professional cover letter generate karti hai.

## Features
- User-friendly input form (Name, Role, Company, Skills).
- Instant cover letter generation.
- "Copy to Clipboard" feature taaki aap letter ko aasani se kahin bhi paste kar sakein.
- Responsive design.

## Kaise Chalayein (How to Run)
1. Is repository ko apne computer par clone karein.
2. `index.html` file ko apne browser mein open karein.
3. Form mein details bharein aur **Generate** button par click karein.

## Technologies Used
- HTML5
- CSS3
- JavaScript
